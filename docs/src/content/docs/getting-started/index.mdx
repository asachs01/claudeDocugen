# Quick Start

DocuGen transforms web workflows into professional documentation with annotated screenshots and contextual explanations.

## Install in 30 seconds

<Tabs>
  <TabItem label="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/asachs01/claudeDocugen/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Windows">
    ```powershell
    irm https://raw.githubusercontent.com/asachs01/claudeDocugen/main/install.ps1 | iex
    ```
  </TabItem>
</Tabs>

## Try it now

In Claude Code or Claude.ai, say:

```
Document this workflow: Create a new GitHub repository
Starting URL: https://github.com/new
```

DocuGen will:
1. Navigate to the URL
2. Guide you through recording actions
3. Capture screenshots at each step
4. Generate markdown with annotated images
